# Statistical Physics of Magnetic Materials

A modern, responsive Flutter application exploring the statistical physics of magnetic materials, including key concepts, student tasks, real-world applications, and model extensions.

## Features

- **Main Concepts**: Outlines fundamental topics like Paramagnetism, Ferromagnetism, Spin systems, Curie law, and Phase transitions.
- **Student Tasks**: Details key educational tasks such as explaining microscopic spins and investigating magnetic ordering.
- **Applications**: Highlights practical uses like Hard drives, MRI, Spintronics, and Magnetic sensors.
- **Possible Extensions**: Suggests conceptual use of the Ising model.

## Tech Stack

- Flutter (Dart)
- Material 3 Design
- Responsive layout for Mobile, Tablet, Desktop, and Web

## Setup and Run

To run this project:

1. Ensure you have Flutter installed.
2. Clone the repository and navigate to the project directory.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to start the app.
