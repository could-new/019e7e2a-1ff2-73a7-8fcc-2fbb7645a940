# Statistical Physics of Magnetic Materials

An educational Flutter application designed to introduce students to the statistical physics behind magnetic materials. The app covers core concepts, outlines student tasks, and provides real-world applications of magnetism.

## Features

*   **Main Concepts:** Learn about Paramagnetism, Ferromagnetism, Spin systems, the Curie law, and Phase transition.
*   **Student Tasks:** Guided tasks including explaining microscopic spin behavior, investigating magnetic ordering, and discussing the temperature dependence of magnetization.
*   **Applications:** Explore how these concepts apply to Hard drives, MRI machines, Spintronics, and Magnetic sensors.
*   **Extension Activity:** Conceptual introduction to the Ising model.

## Tech Stack

*   **Flutter:** Cross-platform framework for building native-compiled applications for mobile, web, and desktop from a single codebase.
*   **Material 3:** Modern, responsive design system.

## Setup & Run Instructions

1.  Ensure you have [Flutter installed](https://docs.flutter.dev/get-started/install).
2.  Clone the repository and navigate to the project directory.
3.  Run `flutter pub get` to fetch dependencies.
4.  Run `flutter run` to launch the application.
